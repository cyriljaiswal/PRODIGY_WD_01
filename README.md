# 🧭 Responsive Landing Page — PRODIGY_WD_01

A responsive landing page featuring a fixed navigation menu that changes style on scroll and highlights items on hover, built with vanilla **HTML, CSS, and JavaScript** for the **Prodigy InfoTech Web Development Internship** (Task-01).

## 🔗 Live Demo
[View Live](#) <!-- replace with your GitHub Pages link after deploying -->

## 📋 Features
- Fixed-position navbar visible on scroll across the whole page
- Navbar background/blur changes once the user scrolls past the hero section
- Animated underline on nav link hover
- Fully responsive — collapses into a slide-in mobile menu on small screens
- Smooth-scroll anchor links to page sections

## 🛠️ Built With
- HTML5 (semantic structure)
- CSS3 (flexbox, media queries, CSS variables, transitions)
- Vanilla JavaScript (scroll listener, mobile nav toggle)
